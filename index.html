Comandos del Bot

  • Prefix: +

    • Comandos Descripción
    +help Lista de comandos
    +invite Invitación del bot
    +servidores Cantidad de servidores del bot
    +stats Estadisticas del bot
    +ping Ping del bot
    +debug Canales de voz en los que esta conectado
    +github Link del repositorio en github

    Comando de Moderación

  • Forma de uso: +ban (Id numérica)

    • Comando Descripción
    +ban Banea permanentemente a un miembro

    Comandos de Música

    Comandos Descripción
    +play Empieza a reproducir una canción/agrega una canción a la lista
    +pause Pausa la canción
    +filter Agrega filtros a la música
    +volume Sube o baja el volumen (0-100)
    +skip Salta la canción actual
    +stop Detener la reproducción de música
    +resume Continuar con la reproducción de música
    +queue Canciones en la lista
    +loop Reproduce la misma canción sin parar en bucle
    +nowplaying Canción reproduciendose actualmente
    +clear-queue Borra la lista de canciones
    +search Busca el nombre de una canción y muesta 19 resultados
    +w-filters Filtros activos e inactivos
    +shuffle Reproducción aleatoria de la lista

    • Porque el bot está offline?

    El bot actualmente se encuentra offline debido a que el host sufrio una perdida de varios nodos (en los que uno de los nodos involucrados era el que alvergaba a chitopanbot), actualmente se esta buscando un nuevo host para poner en funcionamiento el bot de manera estable y sin riesgos de perdidas, el cliente que actualmente se está ejecutando para dar el mensaje del bot offline no tiene habilitado ningún comando ya que el host que se está usando actualmente no tiene un rendimiento optimo para soportar la carga del bot, lamento las molestias.
